# For the Record
**For the Record** is a web project made by Vlad Preda and Alexa Drăgan for the Introduction to Web Programming course taken during the first year of studying at the Faculty of Engineering in Foreign Languages, withing the University POLITEHNICA of Bucharest. Its purpose is to be used for logging and rating music releases, the concept being similar to currently existing applications and/or websites, such as [AlbumOfTheYear](https://www.albumoftheyear.org) or [Letterboxd](https://letterboxd.com). <br>
<br>
The most essential programming language that has been used is **PHP**. Without it, most of the dynamic characteristics would not function this well. Furthermore, **HTML** creates the “skeleton” of the website, representing the base of the web application, and **CSS** embellishes the design of it. <br>
<br>
This project runs with **XAMPP**, by activating Apache and MySQL. The database scheme can be found in this repository under *for_the_record_.sql*, and it only contains the albums that we added as tests.<br>
